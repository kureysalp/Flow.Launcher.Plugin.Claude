# Flow.Launcher.Plugin.Claude

A [Flow Launcher](https://github.com/Flow-Launcher/Flow.Launcher) plugin that opens [Claude AI](https://claude.ai) in your browser with your prompt.

## Features

- Type your prompt and press Enter to launch Claude in your browser
- Incognito mode available as a second result

## Usage

Use the action keyword `cla` followed by your prompt:

```
cla how do I reverse a list in Python
```

## Installation

In Flow Launcher, run:

```
pm install Claude
```
